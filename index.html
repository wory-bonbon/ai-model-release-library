0このファイルの使い方 / 追記方法

運用前提：新モデルのリリース情報（スクショ等）をClaude.aiに渡し、本ファイルも一緒に渡す。Claudeは下記プロトコルに従い、 DATAブロックの MODELS 配列末尾に1件追記して完全版を返す。出力ファイルでローカル保存＆プロジェクトファイルを差し替える。
★お気に入り：各行・各カードの星をクリックでトグル。状態はこのブラウザに保存され、ファイルを差し替えても消えない（idで紐付け）。
▶ AI追記プロトコル（Claudeへの指示。展開して確認）
  1. 編集対象は script 内 === DATA START === から === DATA END === のみ。CSS・描画関数は触らない。
  2. 新モデルは MODELS 配列末尾に追加。既存は消さない。
  3. official_url を web_fetchで再確認してから記入。二次情報のみで status:"official" にしない。
  4. 確認できない項目は推測で埋めず status:"unverified" を維持。last_verified を確認日に。
  5. CHANGELOG 先頭に1行追記。META.session_verified を更新日に。
  6. id は星の紐付けキー。重複しない短い英数字で、一度決めたら変えない。
  7. my_usage（活用例）はユーザー固有。指示なく削除しない。
